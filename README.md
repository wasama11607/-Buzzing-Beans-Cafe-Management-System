# ☕ Buzzing Beans – Café Management System

> A comprehensive **C# Windows Forms Café Management System** developed using **Microsoft Visual Studio**. The application streamlines café operations by providing dedicated interfaces for customers, baristas, and managers while managing memberships, orders, billing, and records through an intuitive graphical interface.

---

# 📖 Overview

Buzzing Beans is a desktop-based Café Management System designed to simplify the day-to-day operations of a coffee shop. The system provides different dashboards for customers, baristas, and managers, allowing efficient management of orders, memberships, billing, employees, and sales records.

The project was developed as part of a university **Software Design & Analysis (SDA)** course and demonstrates practical implementation of Object-Oriented Programming, Windows Forms development, event-driven programming, and file handling in C#.

# 🚀 How to Build and Run

## Requirements

- Windows Operating System
- Microsoft Visual Studio
- .NET Framework
- Required NuGet Packages (automatically restored)

---

## Steps

1. Clone or download this repository.
2. Open **Buzzing_Beans.sln** in Microsoft Visual Studio.
3. Restore NuGet packages if prompted.
4. Build the solution (**Build → Build Solution** or **Ctrl + Shift + B**).
5. Run the application (**Start** or **F5**).
6. Log in according to the required user role.
7. Navigate through the graphical interface to access different café management features.
